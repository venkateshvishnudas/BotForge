For running this project you will need to have the following things:

1. VS Code
2. Python Latest Version


and the following libraries:
pip install streamlit
pip install google-generativeai
pip install python-dotenv
pip install requests

now after installing all these you need to set your own google gemini api key from google studio.
and add the api key in the .env file.


to run the project the following command to be runned:
python -m streamlit run llm.py
